# E-Scooter
## The powerfull electronic scooter with suspension and a 1 hour battery.

## Why its special:
 - has a powerfull motor using a chain link system better than wheel hub system found in mordern scooters.
 - A high capacity battery for extended uses
 - Digital system with throttle, brakes and Battery managment system
 - has a really good custom front wheel suspension.

## Why did I make this?
Hardware projects are fun to make and i wanted something more fun than walking while travelling from one place to another.

## How does it work?
this electric scooter works by using a 24v batte ry pack made from 18650 cells arranged in a 7s 4p configuration which provides the required voltage and enough current to run the scooter. the battery is managed by a bms which protects it from overcharge and overdischarge and balances the cells so the pack stays healthy and also lets it charge . power from the battery goes into the motor controller which regulates how much current flows to the my1020 500W motor depending on throttle input. the motor converts this electrical energy into rotational motion by drivingg the chain which spins the rear wheel and makes it move. the headset and frame hold everything aligned so the power transfer is efficient. the suspension is custom built with 2 cycle supensions attached on a 2 part linkage system, the suspension has enough clearance so that the scooter can easily travel over any bumps and terrain on the way.
